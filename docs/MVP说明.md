# 瑕果智选 AI Agent MVP 说明

## 目标

验证苹果瑕疵果从农户上架到消费者信任购买的最小闭环：图片输入、AI 分级、人工复核、商品说明、反馈回流。

## 当前 AI 形态

当前版本是 mock Agent，不调用真实视觉模型。它使用数据集目录标签或上传时选择的临时标签模拟视觉识别结果，再根据规则表生成结构化 JSON。

## 已跑通能力

- 农户选择样本或上传本地图片。
- 填写产地、重量、采摘时间、期望售价和备注。
- Agent 生成瑕疵类型、等级、置信度、价格建议、农户解释、消费者说明和复核判断。
- 低风险商品进入消费者页。
- 高风险或低置信度商品进入人工复核队列。
- 消费者可模拟下单或提交信任反馈。
- 反馈和人工修正可进入坏例池。
- 可批量运行 12 张样本 Eval。

## 食品安全原则

疑似腐烂、霉变、破皮渗液、低置信度结果必须进入人工复核，不允许直接上架。

## 下一步

1. 接入真实多模态模型。
2. 增加真实农户照片和人工标注。
3. 建立腐烂风险漏判率、可售果误拦截率、消费者信任反馈率等指标。
