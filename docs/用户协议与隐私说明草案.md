# 瑕果智选用户协议与隐私说明草案

更新时间：2026-06-01

## 1. 服务边界

瑕果智选当前版本用于苹果瑕疵果 AI 辅助分级展示、商品草稿生成、人工复核和购买意向收集。当前版本不提供真实在线支付、真实物流发货和正式售后履约。

## 2. AI 分级说明

AI 分级结果仅作为初步参考，不构成最终食品安全检测结论。涉及疑似腐烂、霉变、破皮渗液、低置信度或消费者反馈异常的样本，必须进入人工复核。

## 3. 用户提交内容

用户可能提交以下信息：

- 苹果图片。
- 产地、重量、采摘时间、期望售价。
- 农户备注。
- 消费者购买意向或顾虑反馈。

这些信息仅用于：

- 生成分级建议。
- 生成商品说明。
- 运营复核。
- 坏例复盘。
- 产品改进和质量评测。

## 4. 隐私边界

当前本地 Demo 数据保存在本机浏览器和本地 JSON 文件中。若后续迁移小程序或云端服务，应补充：

- 数据存储位置。
- 数据保存期限。
- 图片用途说明。
- 用户撤回和删除数据方式。
- 联系方式和投诉入口。

当前小程序体验版的数据同步策略为：

- 优先同步本地 API `/api/state`。
- API 不可用时回退微信本地 `wxStorage`。
- 正式上线前需要替换为 HTTPS 后端域名，并补充正式的数据处理和删除机制。

## 5. 交易边界

当前版本中的“模拟下单”和“购买意向”仅用于验证用户兴趣，不代表真实交易成立。正式交易版上线前，需要补充主体资质、商品经营范围、支付、物流、退款和售后规则。
